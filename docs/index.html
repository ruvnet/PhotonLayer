PhotonLayer

Task-trained optics: capture less, decide faster, prove what happened.

This page runs the real PhotonLayer optical simulator compiled to WebAssembly — no server, no mocked results. A preset phase mask transforms the light, it propagates to a sensor, and a small pooled measurement is captured.

how much the sensor averages — bigger = smaller measurement

PhotonLayer stores a learned measurement, not the image. No privacy or security guarantee is claimed. The receipt proves the run is deterministic (the same inputs reproduce the same measurement byte-for-byte) — it is an integrity / anti-swap check, not encryption.
